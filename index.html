https://github.com/rubyonrailstutor/restaurantly

Join me at rubyonrailstutor.com for a fun highly social, distributed real time pair programming class series where we will explore the fundamantels of web application development. We will master the fundamentals of programming tools like ruby, rails, git, haml, heroku, responsive web design, test driven development, vim, sublime, github and the intricacies of software product development. Prior to 2012, I didn't know how to build software. Thanks to Dev Bootcamp and a ton of hard work and support from the San Francisco developer community, in the span of 3 rigorous training months and 18 months working, I now operate as a wholly independent software product tactition. I hope to offer you some tools to make the same transformation.

- John Davison, @railstutor
